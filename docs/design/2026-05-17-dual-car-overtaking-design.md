# 双车协同超车系统设计

## 概述

基于 TI MSPM0G3507 + K230 的双车系统。前车和后车各自独立循迹行驶，后车通过 K230 检测前车 AprilTag 实现自适应距离跟随，通过红蓝信号旗触发超车。

## 系统架构

```
前车 (Lead Car)
┌─────────────────────────────────────────┐
│ K230                                     │
│  ├─ 摄像头 → 红色信号旗色块检测           │
│  └─ UART → MSPM0: "FLAG,1/0\n"          │
│                                         │
│ MSPM0G3507                              │
│  ├─ 灰度传感器 → 循迹                    │
│  ├─ 接收K230数据                         │
│  └─ 蓝牙HC-05 → 后车: "OV\n"             │
└─────────────────────────────────────────┘
                    │
              蓝牙 HC-05 (透传)
                    │
后车 (Rear Car)
┌─────────────────────────────────────────┐
│ MSPM0G3507                              │
│  ├─ 灰度传感器 → 循迹                    │
│  ├─ 蓝牙HC-05 → 接收前车指令             │
│  ├─ UART1 → 接收K230 AprilTag数据        │
│  ├─ 距离PID → 自适应跟随                 │
│  ├─ 超车状态机 → 执行超车                │
│  ├─ 蜂鸣器 → 超车时响                    │
│  └─ OLED → 显示状态                      │
│                                         │
│ K230                                     │
│  ├─ 摄像头 → AprilTag检测                │
│  ├─ 计算距离+角度                        │
│  └─ UART → MSPM0: "TAG,dist=xx,angle=xx"│
└─────────────────────────────────────────┘
```

## UART 通信协议

### K230 → MSPM0（两车相同）

```
TAG,dist=<cm>,angle=<deg>\n
FLAG,1\n   → 检测到红色信号旗
FLAG,0\n   → 信号旗消失
```

- dist=0 表示 AprilTag 丢失
- angle: 负值=偏左, 正值=偏右

### 前车MSPM0 → 后车MSPM0（蓝牙HC-05）

```
OV\n   → 触发后车超车
```

## 后车状态机

```
                   蓝牙收到 "OV"
    ┌──────────────────────────────────────┐
    │                                      │
    ▼                        超车完成       │
┌─────────┐     ┌────────────┐     ┌───────┘
│FOLLOWING│────→│ OVERTAKING │────→┐
│ (默认)   │     │ (蜂鸣器响)  │     │
└────┬────┘     └────────────┘     │
     │ ▲                           │
     │ └───────────────────────────┘
     │     自动恢复跟随
     │
     │ K230 距离数据 → 距离PID → 调速
```

### FOLLOWING 模式

后车循迹行驶，同时根据 AprilTag 距离自动调速：

```
距离误差 = dist - 目标距离(20cm)
距离PID输出 → 修正 base_speed
最终速度 = CLAMP(base_speed(40) + 距离PID输出, 20, 60)
```

| 场景 | dist | 行为 |
|------|------|------|
| 前车太远 | >30cm | 加速到上限(60)追近 |
| 距离适中 | 15-25cm | 正常速度(40)跟随 |
| 前车太近 | <10cm | 减速到下限(20)拉开 |
| 丢Tag | dist=0 | 按 base_speed(40) 跑 |

### OVERTAKING 模式

**触发条件：** 蓝牙收到 "OV" 指令

**开环控制时序：**

| 阶段 | 左轮 | 右轮 | 时间 | 蜂鸣器 | 说明 |
|------|------|------|------|--------|------|
| ① 左偏出线 | 30 | 70 | 300ms | 响 | 向左切出黑线 |
| ② 直行超越 | 60 | 60 | 1500ms | 响 | 两轮同速前进超车 |
| ③ 右偏回线 | 70 | 30 | 300ms | 响 | 向右回黑线 |
| ④ 恢复循迹 | — | — | 自动 | 关 | 切回 FOLLOWING |

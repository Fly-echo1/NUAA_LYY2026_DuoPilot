# DuoPilot v1.0 → v2.0 技术变更文档

> 项目: NUAA 天目启航 · 双车协同循迹系统  
> MCU: TI MSPM0G3507 (Cortex-M0+)  
> 视觉: Canaan K230 (AprilTag 识别)  
> 通信: HC-05 蓝牙 (UART)  
> 日期: 2026-05-17

---

## 一、概述

v2.0 在 v1.0 单车循迹 PID 控制的基础上，引入**双车协同超车**能力。

- **v1.0**: 单车独立循迹行驶，PID 转向 + 速度闭环，OLED 显示与蓝牙调试
- **v2.0**: 前车检测红色信号旗 → 蓝牙通知后车 → 后车 AprilTag 测距跟随 → 收到指令后开环超车

---

## 二、架构变更

### 2.1 系统拓扑

```
v1.0 (单车):
  K230(灰度) ─UART→ MSPM0(控制) ─PWM→ 电机
                          ├─ OLED(显示)
                          └─ HC-05(调试)

v2.0 (双车):
  前车:
    K230(灰度+红色检测) ─UART1→ MSPM0 ─PWM→ 电机
                                       └─ HC-05(发送"OV"指令)
  后车:
    K230(AprilTag测距) ─UART1→ MSPM0 ─PWM→ 电机
                                ├─ HC-05(接收"OV"指令)
                                ├─ OLED(显示距离/状态)
                                └─ 蜂鸣器(超车提示)
```

### 2.2 通讯协议

| 链路 | 协议 | 波特率 | 方向 |
|------|------|--------|------|
| K230 → MSPM0 (前车) | UART1, 文本行 | 9600 | 单向 |
| K230 → MSPM0 (后车) | UART1, `TAG,dist=xx,angle=xx\n` | 9600 | 单向 |
| 前车 HC-05 → 后车 HC-05 | 蓝牙串口, `OV\n` | 9600 | 单向 |

## 三、新增模块

### 3.1 后车状态机 (`Control.h` / `Control.c`)

```
FOLLOWER_FOLLOWING ──收到"OV"──→ FOLLOWER_OVERTAKING ──超车完成──→ FOLLOWER_FOLLOWING
```

- `FollowerState_t`: 跟随/超车 两状态
- `OvertakingPhase_t`: 超车四相位开环控制

### 3.2 K230 AprilTag 数据解析 (`k230_parse_line()`)

解析格式: `TAG,dist=50,angle=-10\n`

- 距离字段 → `g_k230_data.distance_cm`
- 角度字段 → `g_k230_data.angle_deg`
- 无效数据返回 `distance_cm = -1`

### 3.3 UART 环形缓冲区 (`interrupt.c` / `interrupt.h`)

- 128 字节环形缓冲区，中断驱动填充
- `k230_read_line()` — 线程安全行提取，支持临界区保护
- 防溢出: 满时自动丢弃新数据

### 3.4 距离 PID 自适应巡航 (`distance_pid_init/calculate`)

- 目标距离: 20cm（距前车）
- Kp=2.0, Ki=0.05, Kd=0.0
- 积分限幅 ±10, 输出限幅 ±20
- 输出作为左右轮等量速度修正，叠加到转向差速上

### 3.5 开环超车时序 (`overtaking_update()`)

| 相位 | 时间 | 左轮速度 | 右轮速度 | 说明 |
|------|------|---------|---------|------|
| SWERVE_LEFT | 300ms | 30 | 70 | 向左变道 |
| PASS | 1500ms | 60 | 60 | 直行超车 |
| SWERVE_RIGHT | 300ms | 70 | 30 | 向右回原道 |
| DONE | — | STOP | STOP | 恢复跟随 |

超车全程蜂鸣器鸣响。

### 3.6 蓝牙 "OV" 指令检测

- 主循环中轮询 UART0 RX FIFO
- 两字节匹配状态机 (`'O' → 'V'`)，忽略换行符
- 仅在 `FOLLOWER_FOLLOWING` 状态触发超车

## 五、关键 API 说明

### 后车主循环时序（~50Hz）

```
 ┌─ 1. 读取灰度传感器
 ├─ 2. 读取 K230 AprilTag 数据
 ├─ 3. 轮询蓝牙 "OV" 指令
 ├─ 4. 状态机派发
 │    ├─ OVERTAKING → overtaking_update(), 跳转到末尾
 │    └─ FOLLOWING → 继续执行
 ├─ 5. 灰度偏差计算
 ├─ 6. 距离 PID 计算
 ├─ 7. 转向环 PID
 ├─ 8. 目标速度合成 (转向差速 + 距离修正)
 ├─ 9. 速度环 PID (编码器反馈)
 ├─ 10. OLED 显示 (含距离)
 ├─ 11. 蓝牙调试输出
 └─ 12. delay(20ms)
```

### 级联 PID 结构

```
灰度偏差 ─→ 转向环 PID ─→ 差速(左+右-) ─→ 目标速度 L/R
                                              ↓
K230 距离 ─→ 距离 PID ─→ 等量修正(左+右+) ─→ 目标速度 L/R
                                              ↓
                                         速度环 PID(编码器反馈)
                                              ↓
                                            PWM 输出
```

## 六、回归注意事项

1. **SysConfig 无需变更**: UART1、UART0、蜂鸣器 GPIO 均为 v1.0 已配置外设
2. **后车 K230 固件需更新**: 输出 AprilTag 测距协议 `TAG,dist=xx,angle=xx\n`
3. **前车 K230 固件需更新**: 识别红色信号旗后通过 UART 通知前车 MSPM0
4. **前车 MSPM0 固件需更新**: 收到红标志识别结果后通过 HC-05 发送 `OV\n`

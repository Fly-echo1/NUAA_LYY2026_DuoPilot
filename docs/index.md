# DuoPilot — 双车协同循迹系统

## 项目简介

**DuoPilot** 是南京航空航天大学"天目启航"竞赛项目的嵌入式控制系统。系统由**前车**与**后车**两辆智能小车组成，后车可通过灰度传感器循迹行驶，并通过 **K230 AprilTag** 视觉模块测距跟随前车，在收到蓝牙指令后自动执行**开环超车**。

> v2.0 核心能力：双车协同、自适应巡航、指令触发超车

## 核心功能

| 功能 | 说明 | 涉及模块 |
|------|------|---------|
| 灰度循迹 | 8 路灰度传感器检测黑线，加权平均计算偏差 | Grayscale, PID |
| 速度闭环 | AB 相编码器测速，PID 调节 PWM 实现精准控速 | Encoder, PID, Motor |
| 转向控制 | 转向环 PID 输出差速，实现平滑循迹过弯 | PID_turn |
| AprilTag 测距 | K230 识别前车 AprilTag，通过 UART 回传距离/角度 | K230 → UART1 |
| 自适应巡航 | 距离 PID 根据前车距离动态调整行驶速度 | distance_pid |
| 蓝牙遥控指令 | HC-05 接收前车发出的 "OV" 指令触发超车 | UART0 (BT) |
| 开环超车 | 四相位开环控制: 左变道→直行→右回→停止 | overtaking |
| OLED 显示 | 实时显示偏差、PWM、编码器速度、距离 | OLED_Software_I2C |
| 姿态监测 | 9 轴 IMU (预留)，可扩展倾角/航向检测 | WIT, MPU6050 等 |

## 快速开始

### 开发环境

- **IDE**: Code Composer Studio (CCS)
- **SDK**: MSPM0 SDK 2.10.00.04
- **编译器**: TI ARM LLVM 4.0.4 LTS (ticlang)
- **SysConfig**: 1.25.0 / 1.26.2

### 构建

```bash
# 在 CCS 中导入项目
File → Import → CCS Projects → 选择 car.102 目录

# 或使用命令行 (需 CCS CLI)
eclipsec -noSplash -data workspace \
  -application com.ti.ccstudio.apps.projectBuild \
  -ccs.projects car.102
```

### 烧录

在 CCS 中点击 **Debug** 按钮，通过 SWD 接口烧录至 MSPM0G3507。

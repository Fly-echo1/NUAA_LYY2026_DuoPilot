# 控制算法

## 级联 PID 结构

v2.0 采用三级 PID 级联控制，从目标跟踪到电机 PWM 逐层收敛。

```
                      ┌──────────────────┐
  灰度偏差 ──────────►│   转向环 PID      │
                      │  Kp=0.8 Ki=0.005  │
                      │  Kd=0.3           │
                      └────────┬─────────┘
                               │ 差速输出 (-60~+60)
                               ▼
                      ┌──────────────────┐
  K230 距离  ────────►│   距离 PID        │
  目标 20cm           │  Kp=2.0 Ki=0.05   │
                      │  Kd=0.0           │
                      └────────┬─────────┘
                               │ 等量修正 (-10~+10)
                               ▼
                      ┌──────────────────┐
                      │   目标速度合成     │
                      │  L = base + turn  │
                      │      + distance   │
                      │  R = base - turn  │
                      │      + distance   │
                      └────────┬─────────┘
                               │ 目标速度 (0~100)
                               ▼
    编码器反馈 ◄──────────────┐
                      ┌──────────────────┐
                      │   速度环 PID × 2  │
                      │  L: Kp=0.5 Ki=0.01│
                      │  R: Kp=0.5 Ki=0.01│
                      └────────┬─────────┘
                               │ 最终 PWM (0~100)
                               ▼
                      ┌──────────────────┐
                      │   Motor_SetSpeed  │
                      └──────────────────┘
```

## PID 参数表

| PID 实例 | Kp | Ki | Kd | 积分限幅 | 输出限幅 | 功能 |
|----------|----|----|-----|---------|---------|------|
| `pid_turn` | 0.8 | 0.005 | 0.3 | 30.0 | 60.0 | 灰度偏差→差速转向 |
| `pid_speed_L` | 0.5 | 0.01 | 0.1 | 20.0 | 50.0 | 左轮编码器→PWM |
| `pid_speed_R` | 0.5 | 0.01 | 0.1 | 20.0 | 50.0 | 右轮编码器→PWM |
| `pid_distance` | 2.0 | 0.05 | 0.0 | 10.0 | 20.0 | AprilTag 距离→速度修正 |

## 距离 PID (v2.0 新增)

自适应巡航控制，维持与前车约 20cm 距离。

### 控制逻辑

```
error = 当前距离 - 目标距离(20cm)

场景示例:
  实测 30cm → error = +10 → PID 输出正 → 加速追近
  实测 15cm → error = -5  → PID 输出负 → 减速保持
  实测 -1(无效) → 输出 0 → 不干预速度
```

### 参数说明

| 参数 | 值 | 作用 |
|------|-----|------|
| Kp | 2.0 | 距离误差比例响应 |
| Ki | 0.05 | 消除稳态静差 |
| Kd | 0.0 | 不需要微分（距离变化慢） |
| integral_limit | 10.0 | 防积分饱和 |
| output_limit | 20.0 | 最大速度修正量 |
| CLAMP | ±10 | 最终输出再限幅，防止过冲 |

## 转向环 PID

灰度偏差经过转向环 PID 计算得到差速修正值。

### 偏差 → 差速映射

```
偏差 -350 (严重偏左) → PID → turn_output ≈ -280 → L减速 R加速 → 右转
偏差 0 (居中)        → PID → turn_output ≈ 0    → 等速直行
偏差 +350 (严重偏右) → PID → turn_output ≈ +280 → L加速 R减速 → 左转
```

### 目标速度合成

```c
target_speed_L = base_speed(40) + turn_output + distance_adjust;
target_speed_R = base_speed(40) - turn_output + distance_adjust;
CLAMP(target_speed_L, 0, 100);
CLAMP(target_speed_R, 0, 100);
```

| 分量 | 对 L 轮 | 对 R 轮 | 效果 |
|------|---------|---------|------|
| `base_speed` (40) | +40 | +40 | 基础前进 |
| `turn_output` | +turn | −turn | 差速转向 |
| `distance_adjust` | +dist | +dist | 等速加减 |

## 速度环 PID

编码器每 50ms 采样一次脉冲数作为实际速度反馈，20ms 控制周期下同一数值持续 2~3 个周期。

### 控制流程

```
target_speed (0~100)
       │
       ▼
┌──────────────┐
│  误差计算     │  target_speed - encoder_speed
└──────┬───────┘
       ▼
┌──────────────┐
│  PID 计算     │  speed_out = PID(error)
└──────┬───────┘
       ▼
┌──────────────┐
│  合成 PWM     │  motor = target_speed + speed_out
└──────┬───────┘
       ▼
  CLAMP(0, 100) ──► Motor_SetSpeed
```

### 丢线处理

| 连续丢线计数 | 行为 |
|-------------|------|
| 0 (正常) | 正常循迹 |
| 1~10 | 保持上一帧偏差，继续行驶 |
| >10 (~200ms) | 停车 |

## 开环超车控制 (v2.0 新增)

使用 `tick_ms` 全局时间戳实现精确时序，不依赖传感器反馈。

```c
uint32_t elapsed = tick - g_ov_tick_start;

if (elapsed >= OV_TIME_SWERVE_LEFT)   // 300ms 到
    → 切换到 PASS 相位，重置计时器

if (elapsed >= OV_TIME_PASS)          // 1500ms 到
    → 切换到 SWERVE_RIGHT 相位，重置计时器

if (elapsed >= OV_TIME_SWERVE_RIGHT)  // 300ms 到
    → 切换到 DONE 相位，停止电机
```

总超车时间: **300 + 1500 + 300 = 2100ms (约 2 秒)**

# 调试与使用

## UART 调试输出

通过 HC-05 蓝牙 (UART0, 9600 baud) 输出实时调试数据。

### 输出格式

```
dev:+0042 L: 45 R: 38 cnt:2 SL: 34 SR: 30 D: 18
```

| 字段 | 说明 | 示例 |
|------|------|------|
| `dev` | 灰度偏差值 (-350~+350) | `+0042` |
| `L` | 左轮最终 PWM 值 (0~100) | ` 45` |
| `R` | 右轮最终 PWM 值 (0~100) | ` 38` |
| `cnt` | 触发的灰度传感器数量 | ` 2` |
| `SL` | 左编码器速度 (50ms 脉冲数) | ` 34` |
| `SR` | 右编码器速度 (50ms 脉冲数) | ` 30` |
| `D` | K230 测距距离 (cm)，-1 无效 | ` 18` |

### 终端接收

```bash
# Windows (串口助手)
波特率 9600, 数据位 8, 停止位 1, 无校验

# Linux / macOS
screen /dev/ttyUSB0 9600
# 或
minicom -b 9600 -D /dev/ttyUSB0
```

## OLED 显示布局

```
┌────────────────────────────────┐
│ Dev:+0042          (行0: 偏差) │
│                                │
│ L: 45   R: 38      (行2: PWM)  │
│                                │
│ P:  5.1 I: 2.3 D: -0.8 (行4)  │
│                                │
│ SPD:45 D:18       (行6: 速度+距离)│
│ 11011010           (行7: 灰度) │
└────────────────────────────────┘
```

### 超车模式显示

| 元素 | 位置 | 内容 |
|------|------|------|
| 模式标识 | 行0, 偏移 5 字符 | `OV00`~`OV03` |
| 其余行 | — | 不更新 (冻结显示) |

## 蓝牙指令协议

### 前车 → 后车

| 指令 | 长度 | 说明 |
|------|------|------|
| `OV\n` | 3 字节 | 超车触发指令 |

### 后车端处理

```c
// 主循环中轮询 UART0 RX FIFO
while (!DL_UART_isRXFIFOEmpty(UART_BLUETEETH_INST))
{
    uint8_t c = DL_UART_Main_receiveData(UART_BLUETEETH_INST);
    // 两字节匹配: 'O' → 'V' → overtaking_start()
}
```

## K230 AprilTag 协议

### 后车接收格式

```
TAG,dist=50,angle=-10\n
```

### 前车 K230 需输出

前车 K230 检测到红色信号旗后，通过 UART 通知前车 MSPM0：

```
RED_FLAG\n
```

前车 MSPM0 收到后通过 HC-05 发送 `OV\n`。

---

## K230 调试提示

1. **AprilTag 类型**: 建议使用 36h11 家族，尺寸 5~8cm
2. **安装位置**: 后车前部，距地面 5~10cm，朝前略俯
3. **标签位置**: 前车后部垂直张贴，避免遮挡
4. **光照条件**: 200~1000 Lux，避免直射阳光干扰
5. **通讯检查**: K230 上电后应持续输出 `TAG,...` 行，频率约 10~20Hz

## 启动流程

```
上电
 ├─ MSPM0 初始化 (500ms)
 │   ├─ SYSCFG_DL_init()
 │   ├─ SysTick 1ms 中断
 │   ├─ OLED 初始化
 │   ├─ 电机和编码器初始化
 │   ├─ 中断系统启动 (编码器 + K230 UART)
 │   ├─ PID 参数初始化
 │   └─ 蜂鸣器短鸣 (启动提示)
 ├─ OLED 显示 "Car.101 PID" × 500ms
 ├─ OLED 切换到调试布局
 └─ while(1) 主循环开始 (≈50Hz)
```

## PID 调参建议

| 现象 | 应调节参数 | 调节方向 |
|------|-----------|---------|
| 循迹蛇形严重 | 减小 Kd 或减小 Kp | Kp↓ / Kd↑ |
| 过弯速度太慢 | 增大转向环输出限幅 | output_limit↑ |
| 急弯丢线 | 增大 Kp | Kp↑ |
| 电机噪音大 | 减小速度环 Kp | Kp↓ |
| 跟车距离波动 | 减小距离 PID Kp | Kp↓ / Ki↓ |
| 跟车响应慢 | 增大距离 PID Kp | Kp↑ |

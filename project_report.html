📋 项目概览

基于 TI MSPM0G3507 微控制器的两轮差速循迹小车。 通过 8路灰度传感器 实时检测赛道黑线位置， 使用 双环 PID（转向环 + 速度环）实现精确的闭环循迹控制。 编码器提供车轮速度反馈，OLED 实时显示运行数据，蓝牙串口输出调试信息。

▶️ 如何运行

1

安装 Code Composer Studio

版本 v12+，通过 Resource Explorer 安装 MSPM0 SDK 2.10.00.04

2

导入项目

Project → Import CCS Projects → Browse 选择目录

选择 E:\TI\project\new\car.102 → Finish

3

连接调试器

SEGGER J-Link 连接至目标板：SWCLK→PA20 SWDIO→PA19 GND 3.3V

4

构建 & 烧录

构建: Ctrl + B   |   调试: F11 一键烧录
💡
调试提示： 接蓝牙模块 (HC-08/HC-05) 到 PA10 (TX)，PC 端用串口助手查看数据：
dev:+123 L: 45 R: 35 cnt:2 SL: 40 SR: 38
dev=偏差 L/R=左右PWM cnt=传感器数 SL/SR=左右编码器速度

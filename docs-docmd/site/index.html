Car.102 — 智能循迹小车

MCU: TI MSPM0G3507 | IDE: CCS Theia | SDK: MSPM0 SDK 2.09

Car.102 是基于 TI MSPM0G3507 微控制器的智能循迹小车，采用 8 路灰度传感器检测赛道黑线，通过 双路 PID 控制算法实现高速稳定循迹。支持 OLED 显示、UART 调试输出、惯性导航等多种功能。

系统架构

┌─────────────────────────────────────────────────────────┐
│                      Car.102 系统                        │
├─────────────────────────────────────────────────────────┤
│  感知层 │ 灰度传感器 | IMU (BNO08X/660RB/MPU6050)  │
│         │ 超声波 | VL53L0X 激光测距                    │
├─────────────────────────────────────────────────────────┤
│  决策层 │ PID 控制 (转向 + 速度) │ 姿态融合           │
├─────────────────────────────────────────────────────────┤
│  执行层 │ 直流电机 (双路) │ OLED 显示 │ UART 输出    │
└─────────────────────────────────────────────────────────┘

核心功能

功能 说明
循迹控制 8 路灰度传感器检测赛道黑线，PID 计算转向量
双环 PID 转向 PID + 左/右速度 PID，实现平滑循迹
丢线处理 连续丢线保持上次偏差，超过阈值自动停车
状态显示 OLED 实时显示偏差、电机输出、PID 参数、灰度值
调试输出 UART 每秒 50 帧输出运行数据

硬件平台

组件 型号 描述
主控 MSPM0G3507 Arm Cortex-M0+, 80MHz
灰度传感器 8 路数字 8 通道红外反射传感器
IMU BNO08X / IMU660RB / MPU6050 9 轴姿态检测
OLED 0.96" 128×64 软件 I2C 驱动
电机 直流减速电机 双路独立驱动
编码器 霍尔编码器 闭环速度反馈
测距 VL53L0X / 超声波 避障与测距

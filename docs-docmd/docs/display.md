# OLED 显示

## 显示布局

采用 0.96" 128×64 OLED 屏幕，软件 I2C 驱动。

```
┌──────────────────────────┐
│ Dev:  +23    Car.101 PID │  ← 行0: 偏差值 + 项目名
│ L: 45  R: 38             │  ← 行2: 左右电机输出
│ P:  18.4  I:  2.1  D: 6.9│  ← 行4: PID 分量
│ SPD: 45                  │  ← 行6: 左编码器速度
│ 11000111                 │  ← 行7: 灰度传感器状态
└──────────────────────────┘
```

## 多版驱动

项目支持 4 种 OLED 驱动方式：

| 驱动 | 接口 | 适用场景 |
|------|------|----------|
| OLED_Software_I2C | GPIO 模拟 I2C | 任意引脚，通用性最强 |
| OLED_Hardware_I2C | 硬件 I2C | 高速刷新 |
| OLED_Software_SPI | GPIO 模拟 SPI | 高速传输 |
| OLED_Hardware_SPI | 硬件 SPI | 最高性能 |
